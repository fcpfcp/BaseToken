# Base Contract Repo


Basit bir ERC20 token şablonu — Base ağına (testnet veya mainnet) deploy edilebilir.


## Hazırlık


1. Node.js ve npm/yarn kurulu olmalı.
2. `npm install` ile bağımlılıkları yükle.
3. `.env` dosyasını `.env.example` örneğine göre oluştur (PRIVATE_KEY ve PROVIDER_URL).
